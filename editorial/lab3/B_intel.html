有 N 個區間 [L,R] ，每一個區間都有一個價值 vali​。從這 N 個區間選出一組區間集合，這組集合裡面的沒有任何一對區間沒有一個是有重疊的。如何選使得裡面區間的價值和越大越好。

There are N intervals [L,R], and each interval has a value vali​. Select a set of intervals from these N intervals, where no two intervals in the set overlap. How to choose to maximize the sum of values of intervals in this set?

Chinese Editorial

Subtask 1 & 3

所有價值 vali​ 皆為 1，題目等價於如何選擇使得集合裡面的區間數量越多越好。這個問題也被稱為 Activity Selection。

首先對於所有的區間依照 R 由小到大做排序。要讓選擇區間的數量越多越好，就代表盡可能選擇 R 越小的區間。

假設已選擇的區間集合最右端的值為 maxR ，現在要考慮一個區間 [Li​,Ri​]是否能加進集合，他能加進這個集合的條件為 maxR<Li​ ，加進去後 maxR 更新為 Ri​。我們還要在 [Ri​+1,∞) 找出一個組數量最多的區間集合，由於 Ri​ 是目前還沒被考慮過的區間 Ri​的最小值，因此 [Ri​+1,∞) 的範圍會是最大的，所以這樣排序方式會是正確的。

There are N intervals [L,R], and each interval has a value vali​. Select a set of intervals from these N intervals, where no two intervals in the set overlap. How to choose to maximize the sum of values of intervals in this set?

English Editorial

Subtask 1 & 3

All values vali​ are equal to 1, and the problem is equivalent to selecting a set of intervals to maximize the number of intervals in the set. This problem is also known as Activity Selection.

First, sort all intervals in non-decreasing order of R. To maximize the number of selected intervals, choose intervals with smaller values of R.

Assume the rightmost value in the selected interval set is maxR. Now consider whether an interval [Li​,Ri​] can be added to the set. It can be added if maxR<Li​, and after adding it, update maxR to Ri​. We also need to find the maximum number of intervals in the range [Ri​+1,∞), as Ri​ is the smallest value among the intervals not considered yet. Therefore, the range [Ri​+1,∞) will be the largest, and this sorting method will be correct.

Subtask 2

Now consider the interval [Li​,Ri​], and let [Li​,Ri​] be the rightmost of the interval set S. Using a greedy approach, to maximize the value of S, we need to choose the interval with the maximum value from all interval sets, and the rightmost of that set is in [1,Li​−1]. If there exists an interval set T satisfying this condition, then S=T∪[Li​,Ri​], and the value of S is T+vali​.

First, sort all intervals in non-decreasing order of L. Since L is increasing, only consider the maximum value of interval sets for the first i intervals, [Lj​,Rj​] (1≤j<i), and Rj​<Li​. Update the value of S with [Li​,Ri​] as the rightmost.

Subtask 4

The intervals to consider for [Li​,Ri​] will be [0,Li​−1], and since L is increasing, it can be observed that the maximum value will always be increasing.
